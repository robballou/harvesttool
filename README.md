# HarvestTool

Harvest CLI tool.

## Usage

```
# output today's time
harvesttool
```

## Install

Tested in Python 3 but may work in Python 2:

```
pip install -r requirements.txt
```

## Commands

### timesheet

* `timesheet.today`: output today's time
* `timesheet.billable`: output today's billable time
* `timesheet.total`: output today's total time
